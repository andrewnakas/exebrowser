Play Scorched Earth online — the honest answer Won't run here — it's a DOS game, not Windows

Straight answer first: Scorched Earth will not run on ExeBrowser, and we'd rather tell you that up front than waste your time. The legendary "Mother of All Games" — Wendell Hicken's 1991 tank-artillery classic where you lob nukes and Death's Heads across destructible terrain — is a 16-bit MS-DOS program (SCORCH.EXE). ExeBrowser runs Windows .exe files through Wine, and Wine doesn't execute raw DOS binaries — that's a job for DOSBox. So there's no play button here that would actually work. Below is the honest why, where to grab the author's free copy, and the route that does get you playing Scorched Earth in a browser today.

How it works & what to expect

Why it won't run in ExeBrowser

ExeBrowser is Boxedwine — Wine 1.7.55 plus a 32-bit x86 emulator compiled to WebAssembly. It runs Windows programs: Win32 executables, and to a limited degree old 16-bit Windows 3.x apps. Scorched Earth is none of those. It is a pure MS-DOS game, written in Borland C++ and Turbo Assembler in 1991 to talk directly to DOS and a VGA card. There was never a Windows version.

Wine famously "is not an emulator" — it translates Windows API calls. It has no DOS layer of its own; when handed a DOS executable, real Wine simply hands it off to DOSBox. Inside ExeBrowser there is no DOSBox to hand off to, so SCORCH.EXE just won't start. This isn't a slowness or compatibility quirk we can flag as "partial" — it's the wrong kind of executable for this engine entirely. That's why the verdict here is a clean no.

How to actually play Scorched Earth in a browser (DOSBox, not Wine)

The good news: Scorched Earth plays beautifully in a browser — just through a DOS emulator instead of Wine. Several sites wrap the game in js-dos / DOSBox compiled to WebAssembly, which is the correct tool for a DOS title:

  • dos.zone — Scorched Earth — instant in-browser DOSBox play.
  • retrogames.cz — Scorched Earth (DOS) — another browser-based DOSBox build.

Prefer to run it yourself? Download version 1.5 from the author and run it under desktop DOSBox (or DOSBox-X). Mount the folder, type SCORCH, and you're in the menu.

What you're missing (so you know it's worth the DOSBox detour)

Scorched Earth is a turn-based artillery duel for up to 10 tanks on a single 2D battlefield. You set angle and power, fire, and watch the terrain crater away. Between rounds you spend winnings in a shop on absurd ordnance — Funky Bombs, MIRVs, Napalm, Death's Heads, Tracers, parachutes, shields, and the gloriously dumb Riot Charges. It popularized destructible terrain and the "buy weapons between turns" loop that later defined Worms, Pocket Tanks, Gunbound and a thousand others.

It's a tiny, fast, mouse-and-keyboard game that runs perfectly under DOSBox on any modern machine. The only thing it can't do is run on Wine — and now you know exactly why.

Frequently asked questions

Can I play Scorched Earth online in my browser?

Yes — but not through ExeBrowser/Wine. Scorched Earth is a DOS game, so it needs a DOS emulator. Browser builds at dos.zone and retrogames.cz run it in DOSBox-on-WebAssembly with no install.

Why doesn't Scorched Earth work on ExeBrowser when other classics do?

Because it isn't a Windows program. ExeBrowser runs Win32 (and some 16-bit Windows 3.x) executables through Wine. Scorched Earth is a 16-bit MS-DOS binary (SCORCH.EXE), and Wine doesn't execute DOS programs — it relies on DOSBox for that, which ExeBrowser doesn't include.

Is Scorched Earth free, and is it legal to download?

It's free to download from the author. Wendell Hicken offers version 1.5 at no cost on his official site (whicken.com/scorch) with a pay-what-you-want option. That's the legitimate source — skip random abandonware mirrors and get it straight from him.

What's the difference between Scorched Earth and Worms?

Scorched Earth (1991) came first and is a turn-based artillery game with stationary tanks, destructible terrain, and a between-rounds weapons shop. Worms (1995) took that template, added walking units and slapstick. If you like Worms, the DOS original is well worth a DOSBox session.

How do I run Scorched Earth on a modern PC?

Download scorch15.zip from the author, install DOSBox (or DOSBox-X), mount the game folder, and type SCORCH. Or just use a browser DOSBox build like dos.zone for zero setup.
