Play Full Tilt! Pinball online in your browser Partial — the Cadet table can run; launcher, movies and fullscreen are flaky

Full Tilt! Pinball (1995, Cinematronics/Maxis) is the commercial three-table game — Space Cadet, Skulduggery, and Dragon's Keep — that the free Windows Space Cadet was carved out of. Same physics engine, same pre-rendered look, just two more tables and a fancier shell. Because it's built on that same well-behaved 32-bit engine, the individual table executables have a real shot under ExeBrowser — but the retail game wraps them in a DirectDraw launcher and intro movies that don't always cooperate with Wine 1.7.55, so set expectations to "fiddly but playable," not "flawless."

Runs in your browser tab with WebAssembly + Wine — nothing is uploaded. Click the screen to capture input; press Esc to release the mouse.

Full Tilt! Pinball is commercial software. It was developed by Cinematronics and published by Maxis (now part of EA) in 1995, and it has never been released as freeware or placed in the public domain. Only the single Space Cadet table was ever licensed for free distribution — and that was specifically as the cut-down 3D Pinball for Windows – Space Cadet bundled with Windows, not the full retail game.

ExeBrowser does not host Full Tilt! Pinball. To play it legally, use a copy you own from the original retail CD-ROM. We won't link to cracks or "free full version" downloads, and we don't endorse abandonware sites that host copyrighted material without permission.

How it works & what to expect

Full Tilt! ships two builds on the disc: a 16-bit Win 3.x version and a 32-bit Win95 version. The 32-bit build is the one to use here — but its installer is 16-bit and won't run on anything NT-based, so even on a real modern Windows machine people skip the installer and launch the table EXEs directly. Do the same in ExeBrowser.

The good news: the table engine is the same one that powers Space Cadet, which runs well in ExeBrowser. The catch is everything around the table — the FULLTILT.EXE launcher, the pre-rendered intro movies, and the DirectDraw fullscreen mode — which are the parts most likely to show a black screen, a mis-placed window, or to stall under emulation.

Run the tables directly, and skip the movies

Launch a table EXE rather than the launcher whenever you can — it sidesteps the most fragile DirectDraw code. The retail build accepts a -nomovie switch that skips the pre-rendered intro and drops you straight onto the table, which is exactly what you want under emulation. If ExeBrowser lets you set arguments, add -nomovie after the EXE name; if not, just expect the intro to either play slowly or be skippable with a click or Esc.

  • CADET.EXE — the Space Cadet table (best compatibility bet; it's the engine ExeBrowser already runs well)
  • DRAGON.EXE — Dragon's Keep (medieval fantasy table)
  • PIRATES.EXE — Skulduggery (pirate table)
  • FULLTILT.EXE — the menu launcher; prettier, but the riskiest thing to boot first

Each table needs its own data file (CADET.DAT and friends) sitting beside the EXE, which is why you should upload the whole folder, not a lone executable.

Step by step

  1. Get a copy of Full Tilt! Pinball you legally own — the 32-bit Win95 build from the original retail CD-ROM. ExeBrowser does not provide the game.
  2. On the ExeBrowser home page, set the Wine variant to Win32 + Gecko and click Boot Wine.
  3. Use pick a folder and upload the entire game directory so the .DAT table files and movie assets travel with the EXEs.
  4. Choose CADET.EXE as the entry point first — it's the most reliable. If you can pass arguments, add -nomovie.
  5. Click the canvas to capture input. Launch the ball with the Space bar; the flippers are Z and /. Press Esc to release the mouse.
  6. If a table shows only black or stalls, it's almost certainly DirectDraw fullscreen — try the r18 engine variant, or fall back to the lighter free Space Cadet, which uses the same table and is rock-solid here.

Why it's only "partial"

Full Tilt! leans on DirectDraw for its display, and DirectDraw fullscreen is one of the rougher edges of a 1.7.55-era Wine running on a WebAssembly x86 emulator. Even on real Windows 10/11 the game is famous for loading its window half off-screen and mishandling the fullscreen background, so the bar was never high to begin with. Under ExeBrowser you may get a clean Cadet table, or you may get a black launcher, mis-placed window, or a movie that won't advance — it depends on the build and your browser. The pre-rendered tables are not real-time 3D, so the CPU load itself isn't the problem; the display path is. If you want a sure thing, the free Space Cadet extraction of the same Cadet table is the safer, fully-working route.

Frequently asked questions

Is Full Tilt! Pinball the same as 3D Pinball Space Cadet?

Not quite. Full Tilt! Pinball is the commercial 1995 retail game with three tables — Space Cadet, Skulduggery, and Dragon's Keep. Microsoft licensed only the single Space Cadet table and shipped it as the free 3D Pinball for Windows – Space Cadet with Windows 95 Plus!, 98, 2000, and XP. Same engine and same Cadet table; the retail game just has more of everything.

Can I play Full Tilt! Pinball free in my browser?

You can play it in ExeBrowser if you own a copy — but the game itself is not free. It's a copyrighted Maxis/EA title that was never released as freeware, so we don't host it and won't link to cracks. If you just want free pinball in the browser, the Space Cadet table is the legitimately-free option.

Why does it use the Win32 + Gecko variant?

Because it's the same engine as Space Cadet, whose help/registration component can pop a Wine Gecko prompt. Picking Win32 + Gecko bundles Gecko offline so that prompt never interrupts the launch. It does not enable any 3D acceleration — Full Tilt! uses pre-rendered graphics and DirectDraw, not Direct3D.

Why is the verdict only 'partial' and not 'works well'?

The Cadet table can run, but the retail game's DirectDraw launcher, intro movies, and fullscreen mode are flaky under Wine 1.7.55 — issues it has even on real modern Windows. Expect to launch individual table EXEs directly, skip movies with -nomovie, and accept the occasional black screen. For a guaranteed result, use the free Space Cadet page instead.

Why can't I just run the installer?

Full Tilt!'s installer is 16-bit and doesn't run on NT-based Windows at all, let alone in this environment. The fix — on real PCs and here alike — is to skip the installer and launch the 32-bit Win95 table executables (CADET.EXE, DRAGON.EXE, PIRATES.EXE) directly from the game folder.
