Play classic Windows Solitaire online in your browser Works well

The Solitaire that shipped with Windows 3.1 through Windows 7 — sol.exe — is a tiny, classic Win32 game, which makes it a near-perfect candidate for ExeBrowser. So are its siblings: freecell.exe, winmine.exe (Minesweeper) and spider.exe. They boot in seconds, play at full speed, and need nothing modern — no DirectX, no .NET, just plain Win32. Bring your own copy of the EXE (it's bundled with the Windows you already own), load it on the default Win32 variant, and you're dealing cards in under a minute.

Runs in your browser tab with WebAssembly + Wine — nothing is uploaded. Click the screen to capture input; press Esc to release the mouse.

Bring your own copy. Classic Windows Solitaire (sol.exe), FreeCell, Minesweeper and Spider are Microsoft software and are not free to redistribute, so we don't host them. Copy the EXE from a Windows installation you legally own and load it here — ExeBrowser just runs it in your browser.

How it works & what to expect

Classic Solitaire and friends are about as friendly to emulation as Windows software gets. They're small, single-window, software-rendered Win32 programs, so the default range-fetched Win32 variant runs them at full native-feeling speed with their original look intact.

Bring the card graphics

Older versions of Solitaire draw their cards from cards.dll (later cardgames.dll) that lived in the Windows system folder. If you load only sol.exe and it launches with broken or missing card faces, grab that DLL too and upload both together using the pick a folder option, so the game finds its art beside the executable. Bundling the EXE and its DLLs as one folder is the safe default for any of these classic games.

A note on the Windows 3.x version

The original Windows 3.x Solitaire is a 16-bit program. If you specifically want that earliest version, switch to the Win 3.x (16-bit) variant, which handles WIN16 apps that the Win32 variants can't. The Windows 95-and-later sol.exe is 32-bit and runs on the default variant — that's the one most people remember and the one we recommend.

Step by step

  1. Get SOL.EXE (and cards.dll if it's an older build) into a folder. These ship with the copy of Windows you already own.
  2. On the home page, keep the Win32 (default) variant and click Boot Wine.
  3. Use pick a folder, select the folder, choose SOL.EXE, and Run in Wine.
  4. Click the canvas to capture the mouse and start dealing.

Frequently asked questions

Does classic Windows Solitaire run well in the browser?

Yes. sol.exe is a small software-rendered Win32 game, so it boots in seconds and plays at full speed on the default Win32 variant. FreeCell, Minesweeper and Spider Solitaire behave the same way.

My cards show up blank or broken — how do I fix it?

Older Solitaire builds load their card art from cards.dll (or cardgames.dll). Put that DLL in the same folder as sol.exe and load them together with the pick a folder option so the game can find it.

Can I play the original Windows 3.1 Solitaire?

Yes, but it's a 16-bit program, so switch to the Win 3.x (16-bit) variant for that earliest version. The Windows 95-and-later sol.exe is 32-bit and runs on the default variant.

Where do I get sol.exe? Can I download it here?

We don't host it — classic Windows Solitaire is Microsoft software and isn't free to redistribute. Copy sol.exe (and cards.dll) from a Windows installation you legally own, then load it here. ExeBrowser runs the file in your browser; nothing is uploaded to a server.

Will the modern Microsoft Solitaire Collection work?

No. The modern Solitaire Collection is a 64-bit Universal Windows app with online features and DRM, none of which run on Boxedwine's 32-bit engine. Stick to the classic sol.exe from Windows 95 through Windows 7.
