Play classic Windows Solitaire in your browser Works well

The Solitaire that came with Windows 3.1 through Windows 7 — sol.exe — is a tiny, classic Win32 game, and that makes it a near-perfect candidate for ExeBrowser. So are its siblings: freecell.exe, winmine.exe (Minesweeper), and spider.exe. They boot in seconds, play at full speed, and need nothing modern.

AppsSolitaire (sol.exe), FreeCell, Minesweeper, Spider Solitaire
Best Wine variantWin32 (default)
Era1990–2009 — most run on the Win32 engine
How to loadThe EXE plus cards.dll (older versions) — bundle as a folder to be safe
Expected resultFull speed, classic look

Bring the card graphics

Older versions of Solitaire draw their cards from cards.dll (later cardgames.dll) that lived in the Windows system folder. If you upload only sol.exe and it launches with broken or missing card faces, grab that DLL too and upload both together with the pick a folder option so the game finds its art beside the executable.

A note on the Win 3.x version

The original Windows 3.x Solitaire is a 16-bit program. If you specifically want that earliest version, switch to the Wine 3.1 · Win 3.x 16-bit variant, which handles WIN16 apps the Win32 variants don't. The Windows 95-and-later sol.exe is 32-bit and runs on the default variant.

Step by step

  1. Get sol.exe (and cards.dll if it's an older build) into a folder.
  2. On the home page, keep the Win32 (default) variant and click Boot Wine.
  3. Use pick a folder, select the folder, choose sol.exe, and Run in Wine.
  4. Click the canvas to capture the mouse and start dealing.
