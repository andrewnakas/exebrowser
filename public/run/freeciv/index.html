Play Freeciv online in your browser Experimental — older SDL build, may be slow or crash

Freeciv is a free, open-source empire-building strategy game inspired by Civilization. We run the 2.2.7 SDL client (2011) — an older, simpler build with minimal dependencies that works under Wine + WebAssembly. Start a single-player game: click Start new game, configure options, then click Start. Turn-based play means emulation speed barely matters once you're in a game.

Runs in your browser tab with WebAssembly + Wine — nothing is uploaded. Click the screen to capture input; press Esc to release the mouse.

How it works & what to expect

The 2.2.7 SDL client only needs SDL, freetype, and a handful of audio DLLs — no libcurl, no ImageMagick, no Cairo. That simplicity is why it boots here when newer builds don't. The single-player game runs a local server in-process so no network connection is needed to play against AI.

Wine 1.7.55 is old and SDL1 rendering can be unpredictable — you may see rendering glitches or a slow first boot. If you get a blank screen, refresh and try again.

Bringing a newer build

Want a more recent version? Install Freeciv on a Windows PC, copy the full program directory (EXE + data\ folder) into one folder, then upload it with the folder picker. Note: builds from 2.6.x onward use setjmp in a way that crashes Boxedwine — stick to 2.3.x or earlier for best results. Official downloads at freeciv.org/download.

Frequently asked questions

Is Freeciv free to play?

Yes. Freeciv is open-source under the GPL-2.0 license — no DRM, no paywall. We host the 2.2.7 SDL build, which is free for us to redistribute.

Is Freeciv the same as Civilization?

No — Freeciv is an independent free game inspired by the Civilization series. Same style (found cities, research tech, build an empire), but a separate open-source project unrelated to Sid Meier / Firaxis.

Why the 2.2.7 version and not newer?

Newer Freeciv builds (2.6.x+) use setjmp/longjmp for exception handling in a way Boxedwine can't emulate — they crash before the window opens. The 2.2.7 SDL client avoids that pattern and has a much simpler dependency chain that Wine 1.7.55 can handle.

Does Freeciv need an internet connection?

Not for single-player. Freeciv runs a local game server in-process so you play against AI entirely offline in the browser. Online multiplayer is not supported in this sandboxed setup.
