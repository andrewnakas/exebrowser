Play Freeciv online in your browser ▶ Play now — full browser game

Freeciv is a free, open-source empire-building strategy game inspired by Civilization. Click Start Game in the panel above to jump straight into a single-player match against AI. Pick your nation, set the world size, and start building your empire — no download, no install, nothing to upload.

How it works & what to expect

How to play

The game loads via freecivweb.com, the official Freeciv web client. Once the board appears: right-click a tile to move units, left-click cities to manage production, and use the top toolbar for research and diplomacy. Turn-based — press End Turn when done each round.

Want to play locally or bring your own build?

You can also upload a Windows Freeciv installer to ExeBrowser's file picker and run it via Wine + WebAssembly. Builds from 2.6.x onward use setjmp in a way Boxedwine can't emulate, so stick to 2.3.x or earlier. Downloads at freeciv.org/download.

Frequently asked questions

Is Freeciv free to play?

Yes. Freeciv is open-source under the GPL-2.0 license — no DRM, no paywall, no account needed.

Is Freeciv the same as Civilization?

No — Freeciv is an independent free game inspired by the Civilization series. Same style (found cities, research tech, build an empire), but a separate open-source project unrelated to Sid Meier / Firaxis.

Does it need an account?

No. Single-player games launch instantly without registration. You can create a free account on freecivweb.com if you want to save games or play multiplayer.

Who runs the game server?

The game runs on freecivweb.com, the community-run Freeciv web server. ExeBrowser embeds it here for convenience; freecivweb.com is the authoritative host.
