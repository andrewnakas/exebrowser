Play SkiFree online in your browser Works well

The classic SkiFree — Chris Pirih's 1991 Windows skiing game, the one with the yeti that eats you at 2000m — running right in a browser tab. No download for the runtime, no install, no account. SkiFree is a tiny 2D game that draws with plain Windows graphics (no DirectX, no 3D, no networking), so the real Win32 build runs cleanly and at full speed on Wine + WebAssembly. Bring the author's free copy, load it above, and start dodging trees, dogs, and that snowman.

Runs in your browser tab with WebAssembly + Wine — nothing is uploaded. Click the screen to capture input; press Esc to release the mouse.

Bring it from the author's own page

SkiFree was written by Chris Pirih. He makes the game free to download from his official site, but he doesn't publish an explicit license that lets others re-host the file — so we don't ship the bytes here. Grab the free 32-bit build straight from the source: The Most Officialest SkiFree Home Page (ski.ihoc.net). Then use Load your own copy above to run it. Please don't download SkiFree from sketchy "free games" mirrors — the author's page is the safe, original source.

How it works & what to expect

SkiFree is about as friendly as a Windows program gets for ExeBrowser. The 32-bit version (1.04) that Chris Pirih recompiled in 2005 is a small Win32 executable that renders with ordinary GDI graphics — no DirectX, no 3D acceleration, no .NET, no online features. That's exactly the profile that runs well under Boxedwine, and the author himself notes the build "should also work on Wine." Expect smooth, full-speed skiing on any modern laptop.

Use the default Wine variant (32-bit Win32). The much older 16-bit 1.0 release from the Windows Entertainment Pack exists too, but it behaves poorly even on real modern Windows — stick with the 32-bit SKIFREE.EXE for the best experience here.

The controls (and the yeti)

Click the game canvas first so it captures your input, then:

  • ↓ or F — speed up / fast mode (your only hope against the yeti)
  • ← → — steer; hold to carve harder
  • ↑ — slow down / face uphill
  • Space / number keys — tricks off jumps in freestyle
  • You can also steer with the mouse — the skier follows the pointer
  • Esc — release the mouse back to the page

Ski down the endless slope avoiding trees, stumps, rocks and the little dogs. Clear a course (slalom, tree slalom, or freestyle) and you reach the free-skiing stretch — where, at 2000 metres, the Abominable Snowman charges in to gobble you up. Tap into fast mode and you can outrun him for a while; the course loops and he'll hound you for the rest of your run. Outlasting the yeti is the whole legend.

Step by step

  1. Click ▶ Play SkiFree above to boot Wine right on this page.
  2. In a new tab, download the free 32-bit SkiFree from the author's official page and unzip it.
  3. Use Load your own copy → a folder and select the unzipped SkiFree directory (so the EXE finds its files).
  4. Pick SKIFREE.EXE as the entry and click Run in Wine.
  5. Click the canvas to capture input, then steer with the arrow keys or mouse and start your descent.

Frequently asked questions

Can I play SkiFree online for free with no download?

You can play it here with nothing to install — the Wine + WebAssembly runtime streams into your browser. SkiFree itself is a separate free download from author Chris Pirih's official page; grab it once, then load it above and play in the tab.

Is this the real SkiFree or a remake?

It's the real, original SkiFree executable running unmodified. ExeBrowser doesn't re-create the game — it runs Chris Pirih's actual Win32 binary inside an x86 emulator and Wine, drawing to a browser canvas.

How do I beat the yeti in SkiFree?

You can't truly beat him — he's the joke ending of free skiing. After 2000m he charges in and eats you. Press F (or hold ↓) for fast mode the moment he appears and weave through obstacles; skilled players survive long stretches, but the snowman always comes back.

Will SkiFree run well in the browser?

Yes. SkiFree is a tiny 2D game using basic Windows graphics — no DirectX, no 3D, no .NET — so it runs at full speed under Boxedwine. The author even notes the 32-bit build should work on Wine. Use the default Win32 variant and the 32-bit SKIFREE.EXE.

Why isn't SkiFree hosted here for one-click play?

Chris Pirih makes SkiFree free to download but doesn't publish an explicit license letting others re-host the file. To respect his rights we link to his official page instead of shipping the bytes. It's a 10-second download, then it runs right here.
