Play Commander Keen online in your browser Partial — it's a DOS game; a Win32 source port works best

Commander Keen: Marooned on Mars — id Software's 1990 platforming classic, the game whose smooth EGA scrolling helped launch the studio that would later make DOOM. Episode 1 was free shareware then and remains freely redistributable now, so it's legal to play and share. One honest catch up front: the original Keen is a DOS program, not a Windows one, and ExeBrowser runs Win32 software. The reliable way to play it here is a small open-source source port that loads the original shareware data — we explain exactly how below, and we're straight about what does and doesn't boot.

Runs in your browser tab with WebAssembly + Wine — nothing is uploaded. Click the screen to capture input; press Esc to release the mouse.

Commander Keen ep1 is freely redistributable shareware, but the original is a DOS executable which requires a DOS layer Wine here does not support. Use a source port like Commander Genius for the best browser experience — bring your own copy above.

How it works & what to expect

ExeBrowser boots a real copy of Wine on top of a 32-bit x86 emulator compiled to WebAssembly, then runs an actual Windows .exe. That's perfect for Win32 software — but the 1990 Commander Keen was written for MS-DOS, in 16-bit real mode, talking directly to EGA video and the PC speaker. Wine targets Win32, and the Wine 1.7.55 engine here has only thin DOS-machine support (older Wine historically shelled out to DOSBox for DOS programs, a path that isn't reliable in this WebAssembly build). So if you drop the raw KEEN1.EXE in and hit run, it will most likely refuse to start or exit immediately. That's expected, and it's why this page is marked partial rather than a green “just works.”

The reliable way: a Win32 source port

The good news: Keen's community keeps the game alive with open-source source ports — modern rebuilds of the engine that read the original shareware data files and run as ordinary 32-bit Windows programs. Commander Genius is the best-known, an open-source reimplementation that plays the real Marooned on Mars levels. Because a source port is a Win32 EXE, it runs in ExeBrowser the way DOOM source ports do.

  1. Download the free episode 1 shareware from the box above and unzip it.
  2. Grab a Win32 build of an open-source Keen source port (Commander Genius) and unzip it next to the Keen data files.
  3. Use Load your own copy → a folder above and select that combined folder.
  4. Pick the source-port .exe as the entry and click Run in Wine.
  5. Click the canvas to capture input, then play with the arrow keys and Ctrl/Alt.

Keen is a tiny, light game, so when it does run it plays at full speed even with the emulation overhead.

Controls

Classic Keen controls (a source port lets you remap them):

  • ← → — walk left/right
  • Ctrl — jump
  • Alt — use the pogo stick
  • Ctrl + direction — fire the raygun (once you find it)
  • Esc — menu / release the mouse back to the page

If a port uses different defaults, check its in-game options menu.

Honest bottom line

You can play Marooned on Mars in your browser here, legally and for free — but plan to run it through an open-source source port rather than the original DOS KEEN1.EXE, which is unlikely to boot under Win32 Wine. If you'd rather run the genuine DOS binary unchanged, a dedicated DOS layer like DOSBox (or a browser DOS emulator) is the right tool; ExeBrowser is built for Windows executables. Either way, the shareware data is the same and the levels are the real thing.

Frequently asked questions

Can I play Commander Keen online for free with no download?

Yes — Episode 1, Marooned on Mars, was released as free shareware by id Software and Apogee in 1990 and is still freely redistributable. You'll get the best result here by running its data through an open-source Win32 source port; the original DOS launcher is unlikely to boot under Wine.

Why won't the original KEEN1.EXE run in the browser?

Because Commander Keen is a 16-bit MS-DOS program, not a Windows one. ExeBrowser runs Win32 executables on Wine + WebAssembly, and this Wine build has only minimal DOS support. The fix is to use a Win32 source port (like Commander Genius) that reads the same shareware levels.

Is it legal to play Commander Keen here?

Yes, for episode 1 only. id/Apogee distributed Marooned on Mars as shareware meant to be freely copied at no charge. Episodes 2 and 3 of Invasion of the Vorticons are commercial and are not included — buy the full collection from a legitimate store such as Steam.

Is this the real Commander Keen or a remake?

The levels, art, and game data are the genuine 1990 shareware files. To make them run in a Windows-only environment we recommend an open-source source port that re-implements the engine; it plays the original Marooned on Mars maps faithfully rather than re-creating them from scratch.

What's the easiest way to play the original DOS Commander Keen?

If you specifically want the untouched DOS binary, use a DOS emulator such as DOSBox or a browser-based DOS player. ExeBrowser is purpose-built for Windows .exe files, so for Keen here, a Win32 source port is the smoother route.
