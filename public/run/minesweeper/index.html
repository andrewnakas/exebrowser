Play classic Minesweeper online in your browser Runs great — bring your own copy

The real, classic Windows Minesweeper — winmine.exe, the same little grid of mines that ate your lunch breaks for two decades — running right in a browser tab. No download of our stuff, no install, no account. ExeBrowser runs the actual Win32 executable on Wine + WebAssembly. Because Minesweeper is a tiny, plain GDI program with no DirectX, no .NET, and no networking, it's one of the best-behaved things you can run here: WineHQ rates it as running smoothly, and high scores save fine. The one catch is legal, not technical — the game is Microsoft's proprietary software, so you bring your own copy (you almost certainly already own one).

Runs in your browser tab with WebAssembly + Wine — nothing is uploaded. Click the screen to capture input; press Esc to release the mouse.

You need to bring your own copy

Classic Windows Minesweeper (winmine.exe) is proprietary Microsoft software. It shipped bundled with Windows 3.1 through Windows XP/Vista/7 and was never released for free redistribution, so we can't host the file for you. That's fine — the game is a single tiny EXE you almost certainly already own.

The legitimate copy lives at C:\Windows\System32\winmine.exe on any Windows 2000, XP, Vista, or 7 machine you have a valid license for. Copy that file off your own PC and load it above. Don't download winmine.exe from random "EXE fix" sites — they're a common malware vector. If you don't have a Windows copy to hand, the free web clones below are the honest, legal alternative.

How it works & what to expect

Classic Minesweeper is about the friendliest possible workload for Boxedwine. It's a single ~100 KB Win32 executable that draws with plain GDI calls, plays a couple of WAV beeps, and does nothing exotic — no DirectX, no 3D, no .NET runtime, no online services. That's exactly the kind of app this emulator handles well, so it runs at full speed even on modest hardware and feels indistinguishable from native. WineHQ's app database has rated the Windows 2000/XP winmine.exe as running smoothly under Wine for years, including the high-score table.

The version you'll most likely have is the Windows XP build (the blue-titlebar one). The older Windows 9x build works too. Either way it's the same gameplay: left-click to clear a square, right-click to flag a mine, and clear the whole board without detonating one.

Loading your own winmine.exe

  1. On a Windows PC you own a license for (2000, XP, Vista, or 7), copy winmine.exe from C:\Windows\System32\ to a USB stick or folder you can reach.
  2. Click ▶ Play Minesweeper above to boot Wine right on this page.
  3. Use Load your own copy and select your winmine.exe.
  4. Click Run in Wine. The familiar window appears — left-click to clear, right-click to flag, click the smiley to restart.

That's it. There are no extra data files to wrangle: Minesweeper is fully self-contained in the one EXE, which is part of why it's so reliable here.

No Windows copy? Free, legal alternatives

If you don't have a Windows machine to copy the file from, don't grab winmine.exe from a sketchy download site — those are a well-known malware vector. Instead, plenty of free, faithful web clones recreate the classic look and rules in pure JavaScript and run instantly with nothing to install. They aren't the original Microsoft binary, but they play identically and are completely legal to use.

You can also try an open-source desktop clone — there are GPL/MIT Win32 Minesweeper rebuilds out there. Because those are freely redistributable Win32 EXEs, they load into ExeBrowser the same way and skip the licensing headache entirely.
