Chip's Challenge — the deviously good top-down tile puzzler that hooked a generation through the Microsoft Entertainment Pack — running in a browser tab. Guide Chip McCallahan through 149 grid mazes: collect computer chips, dodge fireballs and walkers, slide across ice and force-floors, and reach the exit to advance. The classic CHIPS.EXE is a 16-bit Windows 3.1 program, which is exactly what ExeBrowser's win3x runtime is built for. How it works & what to expect

Which Chip's Challenge is this — and why it runs at all

There are really two PC versions, and the distinction matters here:

  • The Microsoft Entertainment Pack build (CHIPS.EXE, 1992) — the one most people remember from Windows 3.1 / 95. It's a tiny 16-bit Windows program that draws with plain GDI: no DirectX, no 3D, no networking. That simplicity is why it's a realistic target for ExeBrowser. Modern 64-bit Windows can't even run it anymore (Microsoft dropped 16-bit support), so a Wine layer is one of the few ways to play the original.
  • The 2015 Niffler re-release (on Steam) — a modernized 32-bit rebuild. It's the version that's now free-to-keep on Steam, but it's a newer build with its own runtime expectations and is better played natively on a real PC.

ExeBrowser targets the 16-bit classic, loaded through the win3x variant — our 16-bit Windows 3.x mode, made for early-1990s apps exactly like this.

What to expect (the honest version)

Because Chip's Challenge is a turn-based 2D puzzle with no fancy graphics, raw speed isn't a problem — the puzzles don't care that an x86 CPU is being emulated in WebAssembly. The friction is the 16-bit-on-Wine layer:

  • Boots and plays: the board renders, arrow-key movement works, and the level logic is intact. As a puzzle game it's very playable.
  • Rough edges: 16-bit apps under Wine can have palette/redraw glitches, finicky menus, or the occasional refusal to launch. If it won't start cleanly on the first try, that's expected with this era of software.
  • Sound is hit-or-miss: the old MME/PC-speaker audio path may be silent. The game is fully playable muted.
  • No save guarantees: high scores and progress are written to .ini/registry-style files that may not persist between browser sessions. Treat a run as a single sitting.

That's why the verdict is partial, not works great — it's a genuine, playable port of a 30-year-old 16-bit binary, warts and all.

Controls & how to load it

Controls are simple and keyboard-driven:

  • ↑ ↓ ← → — move Chip one tile in that direction
  • Click the game canvas first to capture keyboard input; Esc releases it back to the page
  • Menus (new game, level password, sound) are reachable with the mouse

Steps to play:

  1. Make sure the runtime above is set to the win3x (16-bit Windows 3.x) variant.
  2. Obtain your own copy of the Entertainment Pack CHIPS.EXE (see the licensing box) and unzip it on your computer.
  3. Use “Load your own copy” to drop in the whole folder so CHIPS.EXE finds its companion files.
  4. Click Run in Wine, then click the board and start solving.
  5. If it crashes on launch, try reloading once — and if it still won't go, it may simply be one of the 16-bit titles Wine 1.7.55 doesn't get along with.

If it won't run here

16-bit games are the trickiest category in any Wine-based setup, so have a fallback in mind:

  • Easiest legit option: grab Chip's Challenge 1 on Steam — free to keep, runs natively on modern Windows, and is the official port from the rights holders.
  • Preservation reference: the Internet Archive preserves the Windows 3.x version for study; many browsers can play its in-page emulator there too.
  • Other puzzlers that play nicely here: try Minesweeper or JezzBall for quick GDI-era fixes that tend to behave better than 16-bit binaries.

Frequently asked questions

Can I play Chip's Challenge online for free with no download?

You can run it in your browser here with no install of the emulator — but Chip's Challenge is commercial software, so you must supply your own legally-obtained copy of the game files. We don't host them. The modern Chip's Challenge 1 on Steam is free to keep if you want a no-cost, legal copy.

Is this the original Microsoft Entertainment Pack version?

Yes — ExeBrowser targets the classic 16-bit CHIPS.EXE from the Microsoft Entertainment Pack (Windows 3.1), loaded through our win3x runtime. The newer 2015 Steam rebuild is a separate 32-bit version that's best played natively on a real PC.

Why does Chip's Challenge fail to run on modern Windows but work here?

The Entertainment Pack build is a 16-bit program, and 64-bit Windows removed 16-bit app support entirely. A Wine compatibility layer (like the one ExeBrowser runs in WebAssembly) is one of the few ways to launch the original binary at all.

What are the controls?

Click the game board to capture input, then use the arrow keys to move Chip one tile at a time. Menus are mouse-driven, and Esc releases the mouse back to the page. The goal is to collect the required computer chips, then reach the exit.

Will my progress and high scores save?

Not reliably. The game writes scores and progress to old-style config files that may not persist between browser sessions, so treat each visit as a single sitting. For permanent saves, play the native Steam version.
