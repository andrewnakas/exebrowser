Play classic Windows FreeCell in your browser Runs well — bring your own copy

The FreeCell that shipped with Windows — the calm, all-cards-visible solitaire variant where almost every deal is winnable — is a tiny classic Win32 game, which makes it a great fit for ExeBrowser. The real freecell.exe boots in seconds and plays at full speed on the Wine + WebAssembly runtime, with the familiar green felt and click-to-move cards intact. The one catch is legal, not technical: FreeCell is Microsoft software bundled inside Windows, so we can't host the file — you bring your own copy from a PC you own, then load it right here.

Runs in your browser tab with WebAssembly + Wine — nothing is uploaded. Click the screen to capture input; press Esc to release the mouse.

Bring your own copy — FreeCell is Microsoft software

Classic FreeCell ships inside Windows; it is proprietary Microsoft software and is not free to redistribute, so we don't host it here. That's fine — if you own a Windows PC (or a retired XP/Vista/7 machine), you already own a legal copy. Copy freecell.exe (and its card-art DLL, cards.dll or cardgames.dll) out of C:\Windows\System32 and load that folder above.

If you'd rather not dig through an old install, Microsoft now offers FreeCell free in the modern Microsoft Solitaire Collection on Windows 10/11 — though that Store app is a different, modern build that won't run here. For an instant browser play that we can host, try the open-source card games linked below.

How it works & what to expect

FreeCell is almost the ideal ExeBrowser candidate: a small, pure Win32 game with no DirectX, no 3D, no networking, no .NET, and no DRM. It's the same family as the classic Windows Solitaire and Minesweeper, which we rate Works well — expect the same here. It boots fast and runs at full speed even though an x86 CPU is being emulated inside WebAssembly, because the game barely taxes it.

The classic XP/Vista/7-era build is the one to use. It's a 32-bit executable, so keep the Win32 (default) variant. The much older Windows 3.x FreeCell (from the 1990 Windows Entertainment Pack) is a 16-bit program — if you specifically want that earliest version, switch to the Win 3.x 16-bit variant instead.

Bring the card graphics, too

FreeCell doesn't draw its cards itself — it loads the artwork from a system DLL: cards.dll on Windows XP and earlier, or cardgames.dll on Vista/7. If you copy only freecell.exe and it launches with blank or broken card faces, that missing DLL is why. Grab it from the same C:\Windows\System32 folder, put it beside the EXE, and load the whole folder so the game finds its art.

Step by step

  1. On a Windows PC you own, copy freecell.exe and its card DLL (cards.dll or cardgames.dll) from C:\Windows\System32 into one folder.
  2. Click ▶ Play FreeCell above (or boot Wine from the home page) on the Win32 (default) variant.
  3. Use Load your own copy → pick a folder and select that folder.
  4. Choose freecell.exe as the entry and click Run in Wine.
  5. Click the canvas to capture the mouse, then deal and start moving cards.

Frequently asked questions

Can I play classic FreeCell online for free with no download?

You can play it in your browser here with no install, but you must supply the file yourself. FreeCell is Microsoft software bundled with Windows and isn't free to redistribute, so we don't host it. If you own a Windows PC you already own a legal copy — copy freecell.exe off it and load it above.

Does the old Windows FreeCell run well in the browser?

Yes. It's a tiny pure-Win32 game with no DirectX, 3D, .NET, or networking, so it boots in seconds and plays at full speed on Wine + WebAssembly — the same as classic Solitaire and Minesweeper, which run cleanly here.

Why do the cards show up blank or broken?

FreeCell loads its card artwork from a system DLL — cards.dll on Windows XP and earlier, or cardgames.dll on Vista/7. Copy that DLL from C:\Windows\System32, put it in the same folder as freecell.exe, and load the whole folder.

Where can I legally get the original FreeCell?

From any Windows PC you own — it lives in C:\Windows\System32 on XP, Vista, and 7. Microsoft also offers a free, modern FreeCell in the Microsoft Solitaire Collection on Windows 10/11, but that Store app is a different build that won't run here. We never link to pirated or 'crack' copies.

Which Wine variant should I use for FreeCell?

Use the Win32 (default) variant for the XP/Vista/7 build — it's a 32-bit executable. Only switch to the Win 3.x 16-bit variant if you specifically have the original 1990 Windows Entertainment Pack version, which is a 16-bit program.
