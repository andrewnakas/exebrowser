Play 3D Pinball Space Cadet in your browser Works well

3D Pinball for Windows – Space Cadet is one of the most-missed pieces of bundled Windows software ever — the single pinball table that shipped with Windows 95 Plus!, 98, 2000, and XP before quietly disappearing in Vista. The good news: it's a small, well-behaved 32-bit Win32 program, and it runs in ExeBrowser without much fuss.

App3D Pinball for Windows – Space Cadet (PINBALL.EXE)
Era1995–2001, classic 32-bit Win32
Best Wine variantWin32 + Gecko (the bundled help/registration screen can prompt for it)
How to loadFolder upload — bring the whole pinball directory, not just the EXE
Expected resultPlays at full speed; sound works; smooth on a 2018+ laptop

Why it needs the whole folder

Space Cadet isn't a single self-contained EXE. PINBALL.EXE loads PINBALL.DAT (the table graphics and physics data), PINBALL.MID and WAVE*.WAV for sound, and table-string files from the same directory. If you drop only the EXE, you'll get a window that complains it can't find its table or simply shows nothing. Upload the entire folder using the folder picker so those files sit beside the executable inside Wine's virtual C:\.

Why the +Gecko variant

The original release includes a small HTML-based help and registration component that, under Wine, can pop a prompt to download the Wine Gecko package. Picking the Win32 + Gecko variant bundles that offline so the prompt never appears and the game launches straight into the table.

Step by step

  1. Find a copy of the Space Cadet folder. It shipped on Windows XP under C:\Program Files\Windows NT\Pinball\; copies of the directory are widely archived.
  2. On the ExeBrowser home page, set the Wine variant to Win32 + Gecko and click Boot Wine.
  3. Use pick a folder on step 2 and select the whole pinball directory.
  4. If an entry picker appears, choose PINBALL.EXE, then click Run in Wine.
  5. Click the canvas to capture input. Launch the ball with the Space bar; flippers are Z and /.

What to expect

Space Cadet is light on the CPU, so emulation speed isn't an issue — it plays essentially like the real thing, with working sound. If the table looks dark or input feels stuck, click directly on the canvas to make sure it has focus, and press Esc to release. Nothing you do is uploaded; the files stay in your browser tab.
