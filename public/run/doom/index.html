DOS DOOM vs. DOOM95

The original DOOM.EXE is a DOS program. Wine here targets Win32, so the most reliable thing to run is DOOM95 — the official Windows 95 port id shipped, which wraps the same game in a Win32 executable. If your download contains DOOM95.EXE, that's the entry point (the player above prefers it automatically). A pure DOS build may not start under Wine without a DOS layer; if it refuses, grab a DOOM95 package or use a source port (below).

Even better: a modern source port

DOOM's engine was open-sourced by id in 1997, which spawned dozens of source ports — modernized Win32 rebuilds like Chocolate Doom and Crispy Doom that run the original shareware WAD with fixed resolutions and clean sound. These are small 32-bit Win32 EXEs and tend to run very cleanly in ExeBrowser. Put the source-port EXE and DOOM1.WAD in one folder and load them together.

Frequently asked questions

Can I play DOOM online for free with no download?

Yes. Episode one (Knee-Deep in the Dead) was released as free shareware in 1993 and is still freely redistributable. ExeBrowser runs the real Win32 build in your browser — nothing to install.

Is this real DOOM or a remake?

The real, original executable, running unmodified. ExeBrowser doesn't re-create the game; it runs id Software's actual Windows binary inside an x86 emulator and Wine, rendering to a browser canvas.

What are the controls for DOOM in the browser?

Click the game canvas to capture input. Arrow keys move, Ctrl fires, Space opens doors and operates switches, and the number keys switch weapons. Press Esc to release the mouse back to the page.

Is it legal to play DOOM here?

Yes, for the shareware episode — id distributed it to be freely copied unmodified at no charge. The registered episodes and DOOM II are commercial and aren't included; buy those from id or Steam.

Why might DOOM run slowly in the browser?

An x86 CPU is emulated inside WebAssembly, so you get roughly 10–40% of native speed. DOOM is light enough to play at full speed on most modern laptops; older tablets and Chromebooks may feel slower.
