Play Tyrian 2000 online in your browser Works — free freeware classic runs via DOSBox in your browser

Tyrian 2000 — the beloved 1995/1999 vertical-scrolling space shooter from Eclipse Software and Epic MegaGames — is genuinely free: the rights holder released the full game as freeware. And now it plays directly in your browser via DOSBox compiled to WebAssembly. Hit play — the original DOS game boots right here, no install, no port needed.

How it works & what to expect

There are two layers to be honest about here: the license and the binary.

The license is the easy part. Author Jason Emery released the complete Tyrian 2000 as freeware in 2004, and the artwork was later placed under an open-content license. The data files (graphics, levels, music) are freely redistributable, and the game is also offered free of charge on GOG. So there's nothing shady about playing it — no piracy, no cracks, no commercial copy required.

The binary is the tricky part. The original TYRIAN.EXE / TYRIAN2K.EXE is a DOS executable. ExeBrowser's Wine targets 32-bit Win32, so a pure DOS binary will not start cleanly without a DOS layer. That's why the verdict here is partial: the game is free and the gameplay is timeless, but getting it running in a browser Win32 environment takes the right approach.

The reliable path: the OpenTyrian2000 port

The cleanest way to run Tyrian 2000 in ExeBrowser is the open-source OpenTyrian2000 source port — a faithful, native rebuild of the game that ships as a real Win32 executable rather than a DOS one. You load the port's .exe alongside the freeware Tyrian 2000 data files and it plays the original game, levels and all.

  • Grab an OpenTyrian2000 Windows release (it's GPL-licensed and free).
  • Grab the freeware Tyrian 2000 data (the tyrian2000.zip set of .dat, .shp, .lvl files).
  • Put the data files in a data\ folder right next to the port's EXE, then load the whole folder above.

Honesty note: OpenTyrian2000 is SDL-based, and SDL plus the Wine + WebAssembly layers can be demanding — expect it to run, but a heavy SDL2 build may be slower or fussier than a lightweight SDL1 one. If one build stutters, try another release, or the r18 variant if the default crashes on launch.

Frequently asked questions

Is Tyrian 2000 really free to play?

Yes. The full game was released as freeware by the rights holder in 2004, the graphics were later put under an open-content license, and it's also given away free on GOG. There's nothing to buy and nothing shady about playing it.

Can I play Tyrian 2000 in the browser with no download?

Sort of — you play it in your browser via Wine + WebAssembly, with no install, but you do need to load the game files into the player. Because Tyrian is a DOS game, the smoothest route is the open-source OpenTyrian2000 Win32 port plus the freeware data files, loaded with the on-page uploader.

Why doesn't the original Tyrian EXE just run?

The original Tyrian / Tyrian 2000 executable is a DOS program. ExeBrowser's Wine targets 32-bit Win32, so a pure DOS binary won't start cleanly without a DOS layer. The OpenTyrian2000 source port is a native Win32 rebuild, which is why it's the recommended way to play here.

What is OpenTyrian2000?

It's a free, open-source (GPL) port of Tyrian 2000 that recreates the original game as a modern, native executable using the freeware Tyrian 2000 data files. It's the most reliable way to run the game inside ExeBrowser's Win32 environment.

Will it run at full speed?

Maybe. ExeBrowser emulates an x86 CPU inside WebAssembly at roughly 10-40% of native speed, and OpenTyrian2000 is SDL-based, which adds overhead. The game is light enough that a modern laptop often handles it fine, but expect possible slowdown on older tablets or Chromebooks.
